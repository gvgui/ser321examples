You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- accepts two integers as inputs and returns the product
    --- ex: /multiply?num1=3&num2=4
  • /github -- accepts a query and runs it with Github's REST API
    --- returns the repo name, id, and owner name of each repo
    --- ex: /github?query=users/amehlhase316/repos
  • /fruitOrVeg -- enter a type (fruit or vegetable) and a color to see an image!
    --- choose between the following colors: red, orange, yellow, green, blue, purple
    --- there are multiple images for type and color pairs so refresh the page if you'd like to see that
    --- ex: /fruitOrVeg?type=fruit&color=purple
  • /bored - enter a number of participants and a type of activity to receive an activity suggestion with a price approximation
    --- Bored api was used for this request, so there are not many suggestions for activities with participants greater than 1
    --- Activity Types Include: relaxation, recreational, education, diy, music, cooking
    --- ex: /bored?participants=5&type=music
